# 🚀 Python GUI Mastery: Management & Utility Suite

Welcome to the **Python GUI Mastery** project! This repository is a comprehensive collection of professional desktop applications built using Python and the Tkinter library. It serves as both a practical toolset for management and a robust learning resource for GUI development.

---

## 🌟 Key Application Suites

### 🎓 Academic Management System
A full-featured system for managing an academic institution's core data:
- **Professor Management**: Add, search, modify, and delete professor records with a dedicated GUI.
- **Module Management**: Track academic modules, their specialties, and requirements (Obligatory vs. Optional).
- **Integrated Menu**: A unified interface to launch specific management sub-apps.

### 💼 Employee & Payroll Manager
Streamline staff management and salary calculations:
- **Employee Database**: Detailed records including Reference, Name, City, Gender, and Level.
- **Salary Checkers**: Quick tools to verify salary scales and legal requirements.
- **Payroll Utilities**: Experimental tools for payroll processing.

### 👥 Client Relationship Manager
Keep track of your clients with ease:
- **Client Profiles**: Dedicated models for client data (Name, Age, Gender).
- **Search & Filter**: Efficiently retrieve client information from the database.

### 🧮 Calculator Suite
From simple math to modern interfaces:
- **GUI Calculator**: A sleek, black-themed desktop calculator with advanced layout.
- **Math Utilities**: Backend scripts for complex mathematical operations and equation solving.

### 📚 Learning & GUI Components
A treasure trove of examples for developers:
- **GUI Basics**: Ready-to-use snippets for Checkboxes, RadioButtons, Listboxes, and Menus.
- **Core Basics**: Exercises for loops, strings, functions, and exceptions.

## 🛠️ Getting Started

### Prerequisites
- Python 3.10 or higher
- Tkinter (Usually included with Python)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/charaf12-u/Python-GUI-Mastery.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Python-GUI-Mastery
   ```

### Running the Applications
You can launch any of the main applications from the root directory:
```bash
# Launch the Academic Menu
python academic_management/main_menu.py

# Launch the Employee Manager
python employee_management/main.py

# Launch the GUI Calculator
python calculators/gui_calc.py
```
